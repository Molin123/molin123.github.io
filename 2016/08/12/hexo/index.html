如何在githuhb上面搭建一个博客

前期准备

  1. 安装git
  2. github账号
  3. 安装hexo脚手架

搭建步骤

创建项目

在github上创建一个新的仓库，名字格式为 XXXX.github.io 。XXX必须是你的用户名。

安装hexo-cli

初始化项目

  1. git init
  2. hexo init
  3. 更改配置文件_config.yml如下：

注意：key和value之间必须有一个空格

  1. npm install hexo-deployer-git --save
  2. hexo generate
  3. hexo deploy（这个过程很慢，请耐心等待）
    第一次会提示输入github账号和密码

发布文章

  1. 执行 hexo new + 文章名，然后就可以去resource/_posts下面编写对应的md文件就可以了
  2. hexo generate(相当于add+commit)
  3. hexo deploy(相当于push)
